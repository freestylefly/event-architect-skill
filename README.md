# Event Architect Skill —— 活动方案策划智能助手

> 活动的成功 = 30% 的创意 + 70% 的执行细节。让 AI 帮你搞定那 70%。

## 这是什么？

`event-architect` 是一个 AI Agent Skill，专为活动策划场景设计。它能从一句话需求出发，自动生成包含创意方向、执行流程、预算明细、物料清单和应急预案的完整策划案。支持团建、年会、发布会、沙龙、展会、内部培训等多种活动类型。

## 核心特性

- 🎯 **四阶策划法**：定目标 → 出创意 → 排执行 → 防翻车，每一步都有标准化输出
- 📋 **六大活动模板**：团建、年会、发布会、沙龙、展会、内部培训——各有专用策划框架
- 💰 **预算智能分配**：根据活动类型自动推荐预算比例，精确到每一项明细
- ⚠️ **风险预判引擎**：自动扫描天气、设备、人员、超时、预算五大风险，提前准备 Plan B
- 📅 **倒排时间线**：根据活动日期反推准备节点，任务精确到人和截止日期
- 📦 **物料清单生成**：规格、数量、采购渠道、最晚到货日，一张表搞定采购

## 快速开始

### 安装

```bash
git clone https://github.com/freestylefly/event-architect.git
```

将项目放入你的 AI Agent 的 Skills 目录即可。

### 使用

对 Agent 说：

```
帮我策划一个团建方案：
预算 3000 元，大概 15 个人，在北京市区内，要有趣一点，不要单纯吃饭。
```

Agent 会自动匹配"团建活动"模板，提供 3 个创意方案，生成包含流程、预算、物料和应急预案的完整策划案。

### 更多用法

```
# 新品发布会
帮我策划一场新品发布会：产品是智能手表，预算 5 万，邀请 50 位媒体。

# 只要预算表
帮我做一份年会的预算明细表，总预算 8 万，200 人规模。

# 细化流程
这个方案的流程部分再细化一下，每个环节精确到分钟。

# 倒排时间线
活动定在 5 月 20 日，帮我做一个倒排准备时间线。
```
